# Music Student MVP

App minimalista para estudiantes de música.  
Permite subir audio (MP3/MP4) y partitura (PDF/JPG/PNG), reproducir el audio y ajustar un desfase (offset).

✅ **Etapa 1: MVP Básico**  
- Subida de audio  
- Subida de partitura  
- Reproducción  
- Control de offset (sincronización manual)

## Cómo usar
1. Clona este repositorio
2. Abre `index.html` en tu navegador
3. ¡Sube tu audio y partitura!

> ⚠️ Esta versión no sincroniza automáticamente. El offset es solo informativo (para MVP).  
> La sincronización visual/automática se añadirá en versiones futuras.

Hecho con ❤️ para estudiantes de música.
